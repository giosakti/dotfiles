# gio's dotfiles

## How to Apply

1. Clone and `cd` into the repository
2. `git submodule update --init --recursive`
3. `./install.sh`

Profit.

## Adding new Submodules

1. `mkdir -p <path>`
2. `git submodule add <url> <path>`
